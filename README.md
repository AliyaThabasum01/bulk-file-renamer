# 📂 Bulk File Renamer

A simple Python CLI tool to rename all files inside a folder.

## ✨ Features

- Rename multiple files at once
- Keep original file extensions
- Custom filename prefix
- Lightweight and beginner-friendly

## ▶️ Run

```bash
python main.py
```

## 📌 Example

Before:

```
IMG_001.jpg
IMG_002.jpg
IMG_003.jpg
```

After:

```
Vacation_1.jpg
Vacation_2.jpg
Vacation_3.jpg
```
